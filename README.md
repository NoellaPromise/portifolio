
# Personal Portfolio

This is my modern, responsive portfolio website built with Next.js, Tailwind CSS, and TypeScript. This project showcases my skills, projects, resume, and contact information.

## Features

- Built with Next.js 13+ (App Router)
- Styled using Tailwind CSS
- TypeScript for type safety
- Light/Dark theme toggle
- Fully responsive design
- Modular, reusable components
- About, Projects, Resume, and Contact sections
- Contact form with validation
- Easy deployment on Netlify

## Getting Started

1. **Install dependencies:**
	```bash
	npm install
	```

2. **Run the development server:**
	```bash
	npm run dev
	```
	Open [http://localhost:3000](http://localhost:3000) in your browser.

3. **Customize your content:**
	- Edit pages in `app/`
	- Update components in `components/`
	- Add images to `public/Images/`

## Project Structure

- `app/` — Main pages and layouts
- `components/` — Reusable UI components
- `lib/` — Utilities and type definitions
- `public/` — Static assets (images, icons)
